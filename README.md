# 🍽️ 리뷰이벤트 플랫폼

요식업 매장용 게임형 리뷰 이벤트 시스템입니다. 고객들이 리뷰를 남기고 게임에 참여해 쿠폰과 상품을 받을 수 있습니다.

## ✨ 주요 기능

### 👥 고객 화면
- **로그인**: 전화번호 + 닉네임으로 간단 가입
- **메인 게임 3종**
  - 🎡 룰렛: 매일 1회 참여
  - 🃏 카드뽑기: 게임방식 선택 가능
  - 🎟️ 긁는복권: 동적 결과 공개
- **추가 이벤트**: 카카오맵/구글/티맵 리뷰 연동
- **VIP 등급**: 브론즈~다이아(5단계)
- **스탬프 카드**: 고객별 독립 카운트
- **만족도 설문**: 별점/매트릭스/객관식/주관식
- **쿠폰 함**: 자동 발급 쿠폰 관리
- **네이버 플레이스**: 평가 연동

### 🏪 매장 관리자 화면
- **통계**: 일/주/월별 참여 현황, VIP 분포
- **고객 관리**: 연락처 CSV/TXT 다운로드
- **QR코드**: 매장별 고유 QR 생성
- **게임 설정**: 게임방식, 상품, 확률 커스터마이징
- **쿠폰 발송**: CRM 문자/복사 기능
- **설문 관리**: 5가지 문항 유형 + 응답 통계
- **추가이벤트**: 플랫폼별 독립 설정
- **매장 설정**: 배경이미지, 소셜채널, PIN 보호

### 👨‍💼 총괄 관리자
- **전체 매장 현황**: 신호등 상태(7일/14일/15일+) + 휴면순 정렬
- **일괄 관리**: DB CSV 다운로드
- **대신 관리**: 사장님 대리 세팅

## 🚀 빠른 시작

### 로컬 개발
```bash
npm run dev
# 또는
python -m http.server 8420
```

브라우저에서 http://localhost:8420 열기

### Vercel 배포
상세 가이드는 [DEPLOY.md](./DEPLOY.md) 참고

```bash
# 방법 1: GitHub + Vercel 자동 배포 (권장)
git init
git add .
git commit -m "Initial commit"
git push -u origin main
# → Vercel에서 저장소 연결 후 자동 배포

# 방법 2: Vercel CLI
npm install -g vercel
vercel
```

## 🔧 설정

### 매장 기본 정보 (매장 관리자 화면)
- 매장명, 주소, 전화번호
- 배경이미지 업로드
- 사장님 PIN 보호
- 네이버 플레이스/카카오채널 URL

### 게임 설정
- **상품**: 최대 8개, 상품명 + 혜택 + 확률
- **게임방식**: 메인은 1가지 택1, 추가이벤트는 플랫폼별 독립
- **VIP 등급**: 방문 회수 기준(10/20/30/50) + 자동 쿠폰
- **스탬프**: 목표 개수, 보상, 이미지 업로드
